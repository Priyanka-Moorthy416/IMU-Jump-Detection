# IMU-Jump-Detection
Developed an IMU-based jump event detection system using embedded sensors and Finite State Machine (FSM) logic to accurately identify jump phases &amp;The system analyzes real-time motion data to estimate flight time and jump height, improving movement tracking accuracy and for sports performance analysis and real-time activity monitoring applications.
An embedded system project developed using IMU sensors and Finite State Machine (FSM) logic to accurately detect jump phases such as take-off, flight, and landing.

## Features
- Real-time jump detection
- Flight time estimation
- Motion tracking and analysis
- FSM-based phase identification

## Technologies Used
- Arduino
- Embedded C
- IMU Sensor
- FSM Logic

## Applications
- Sports performance analysis
- Activity monitoring systems
- Motion tracking applications
